LETITBE.txt: an AI-powered lyricist

LETITBE.txt is an AI-powered lyric generator, which utilizes machine learning insights from the Modzy Graph Embeddings to predict and generate realistic lyrics from The Beatles' full discography.

How it works:

LETITBE.txt's python scripts search genius.com, a lyric annotation website, using the Genius API to retrieve URLs for every Beatles song. (anthologies and collaborations excluded). Then, each song webpage's HTML is scraped using BeautifulSoup4 to extract the plaintext lyrics.
After formatting out excess markup text, all song lyrics are then fed through a word-processor algorithm to generate a "Markov Chain": a sequential representation of existing word-to-word pairings as found in the input lyric data. Example Image of Markov Chain here. Markov Chains are powerful models for generating natural grammatical sentences. Without an underlying ruleset for grammar and semantics, they can create natural, understandable sentences that aren't contained in the original dataset!
